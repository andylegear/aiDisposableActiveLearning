# Surveys

This directory holds survey-related materials for the study.

## Structure

```
surveys/
  expert-reviewer-invitation.md     # Template email/message for recruiting expert reviewers
  coordinator-invitation.md         # Template email/message for recruiting module coordinators
  consent-information.md            # Information sheet for reviewers (if needed)
```

## Notes

- The actual evaluation protocol documents are documented in the paper.
- Completed survey responses (transcribed to JSON) go in `data/evaluations/`.
- This directory holds the *distribution* materials, not the *collected* data.
